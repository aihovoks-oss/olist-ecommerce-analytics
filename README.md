# olist-ecommerce-analytics
# 📊 Бизнес-аудит бразильского маркетплейса Olist (e-commerce)

Глубокий аналитический аудит операционных и финансовых показателей маркетплейса Olist (бразильский аналог Ozon/Wildberries). В рамках проекта была проведена сквозная аналитика пяти связанных баз данных с целью выявления ключевых точек потери клиентской лояльности и оптимизации товарной матрицы.

## 🎯 Ключевые инсайты и результаты проекта

### 1. Географический кризис логистики
Анализ временных рядов доставки выявил критическую неравномерность распределения логистических мощностей по стране. В то время как в центральных штатах (например, Сан-Паулу) доставка укладывается в рыночные стандарты, в удаленных регионах (лидер — штат **Рорайма / RR**) средний срок ожидания составляет **до 29 дней**. 

### 2. Метрика «Обещание vs Реальность» (Фокус на психологии клиента)
Была рассчитана кастомная метрика `delivery_delay` (число дней просрочки относительно обещанной клиенту даты при оформлении заказа). Выявлена жесткая математическая зависимость:
* Заказы с оценкой **5 звезд** доставляются с нулевой просрочкой (в среднем 0.13 дня).
* Заказы с оценкой **1 звезда** имеют среднее опоздание в **4.5 дня** от обещанного срока.
* **Вывод для бизнеса:** Негатив клиентов вызван не длительностью доставки как таковой, а нарушением обязательств. Решением является повышение точности прогнозных алгоритмов на сайте, а не физическое ускорение транспорта.

### 3. Двусторонний Парето-анализ (Локализация хейта)
Применение закона Парето к неудовлетворенности клиентов показало, что **всего 15 категорий товаров генерируют 80% всех плохих отзывов (1-2 звезды)** на платформе. 
* Абсолютные лидеры по генерации негатива — категории `cama_mesa_banho` (постельное белье) и `moveis_decoracao` (домашний декор), дающие суммарно **более 21% всех отрицательных оценок**.

### 4. Финансовый ABC-анализ ассортимента
Проведена классификация товарных категорий по их вкладу в общую выручку компании:
* **Группа A (16 категорий):** Генерирует **79.1% всей выручки** — экономическое ядро бизнеса.
* **Группа B (16 категорий):** Приносит **15.7% выручки** — стабильный средний класс.
* **Группа C (41 категория):** Формирует «длинный хвост», приносящий всего **5.2% выручки**.
* **Рекомендация:** Перевод 41 категории группы C на модель работы FBS (продажа со склада продавца) для разгрузки складов и перенаправление операционного фокуса на оптимизацию логистики группы A.
